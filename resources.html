Authentication & Cloud Sync — How It Works

Strat Planner Pro uses a sidebar navigation with real-time online/offline status indicators and visual cues distinguishing cloud-synced (authenticated) from local-only mode. Your BARMM investment project is safely stored whether you're online at the venue or working offline.

☁️
● Cloud Synced Mode
Authenticated login. All project data synced to cloud. Collaborate across team members. Access from any device. Version history preserved.
💾
◆ Local-Only Mode
No login required. Data stored locally. Full functionality available offline during workshop. Sync to cloud when connectivity resumes.
